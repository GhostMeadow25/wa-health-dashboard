Washington Children's Health Insurance Coverage Dashboard: An interactive R Shiny dashboard visualizing uninsured rates among children under 19 across all 39 Washington State counties, built with 2024 ACS 5-Year Estimates data from the U.S. Census Bureau.

Overview: This dashboard was built as a personal portfolio project to explore children's health insurance coverage disparities across Washington State. It allows users to: Select any county or the statewide total and view a summary data card with the uninsured rate, count, and comparison to the state average Explore a choropleth map of all 39 counties colored by uninsured rate Compare all counties side-by-side in a ranked bar chart, with counties above the state average highlighted in red

About: Built by Lily Meadow · B.S. Data Analytics, Eastern Oregon University (2025) Interested in data for social good — community health, and environmental sustainability.
